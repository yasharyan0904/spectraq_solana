  1. Pyth staleness check (safety.ts::checkPythStaleness) вҖ” if price > 60s old,
  skip.
  2. Pull recent prices вҖ” Binance primary, Pyth Hermes fallback (priceFeed.ts).
  3. Compute signal:
    - MOCK_MPC=true вҶ’ MA crossover in TypeScript вҶ’ call mock_callback_signal
  on-chain to stamp signal.
    - MOCK_MPC=false вҶ’ encrypt prices with x25519 вҶ’ submit to Arcium вҶ’ wait for
  Ready callback.
    - Result is -1 / 0 / 1.
  4. Read live vault balances (trader.ts::readVaultBalances).
  5. decideTrade(signal, position, balances) вҖ” pure function:
    - signal=1 + position=usdc вҶ’ BUY (USDCвҶ’SOL, label long_open)
    - signal=0 + position=sol вҶ’ SELL (SOLвҶ’USDC, label long_close)
    - any other combination вҶ’ no-churn, return null.
  6. executeTrade(action) вҖ” quote Jupiter вҶ’ fetch swap-instructions вҶ’ build the
  Anchor executeTrade ix вҶ’ CPI into Jupiter from inside the vault PDA. Vault
  program enforces 30 % cap and 5 % oracle slippage floor.
  7. settle_pnl вҖ” refresh cached vault balances & emit TradeExecuted event.

  The dashboard's "Recent trades" table reads TradeExecuted events (Anchor logs,
